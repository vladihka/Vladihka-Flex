What is FlexBox?

Flexbox is an entire module that contains a set of properties and their values for both the parent flex-container and its child flex-elements.

The main advantage of flexible FLEX markup is the ability to change the width/height of the space around and the order of the flex elements in order to best fill the available flex container space.
